## 今あるやつ
### C

- UnionFind.c: UnionFind木
- avltree.c: AVL木（非再帰）
- avltree_rec.c: AVL木（再帰）
- bsort.c: ボゴソート
- convolution.c: 畳み込み
- fenwicktree.c: Fenwick木 (Binary Indexed Tree)
- fft.c: FFT
- maxflow.c: 最大流
- mergesort.c: マージソート
- rbt.c: 赤黒木
- segtree.c: セグメント木

### C++

- BIT.cpp: Binary Indexed Tree
- UnionFind.cpp: UnionFind木
- modint.cpp: modint
- segtree.cpp: Segment Tree

### Brainf*ck

- increment.bf: マルチバイト整数(符号無し)のインクリメント
